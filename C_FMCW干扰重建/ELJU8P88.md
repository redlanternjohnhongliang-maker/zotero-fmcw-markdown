## ABSTRACT

In this paper, a novel interference mitigation approach using an autoencoder in combination with a traditional interference detection filter is introduced. It is shown that by employing the gated convolution, the encoder has the ability to learn the signal pattern from the remaining interference-free signal. The decoder can recover the interference-contaminated signal segments from the bottleneck representation as computed by the encoder. Experimental results show that the proposed method can provide a remarkable improvement in signal-to-interference-plus-noise ratio (SINR) and preserves its robustness on real radar measurements in severely disturbed scenarios that are more complex than the training dataset.

Index Terms— automotive radar, interference mitigation, autoencoder, deep learning

## 1. INTRODUCTION

Automotive radar can provide reliable safety support for modern vehicles under almost all weather conditions and thus plays an important role in advanced driver assistant systems. However, the mutual interference between automotive radars can raise the noise level and reduce the signal-to-noise ratio [1–3]. Therefore, the performance of automotive radar sensor can be affected if no countermeasures are taken [4, 5].

Various signal processing techniques have been proposed to address the problem of mutual interference [6–14] in the frequency-modulated continuous wave (FMCW) radar or the chirp sequence (CS) radar [15]. Besides the traditional signal processing approaches, recent research results have shown that deep learning approaches can be used to solve this problem [16–18]. A recurrent neural network (RNN) model with self attention mechanism is trained in [16] and applied to suppress the interference-contaminated signal in the time domain. In [17], convolutional neural networks (CNNs) are employed for noise suppression of both the range profile and the range-Doppler (RD) spectrum, where the range profile can be obtained after applying the first fast Fourier transform (FFT) along the fast-time samples and the RD spectrum can be obtained after applying the second FFT to the range profile. However, the performance of the CNNs used in [17] may be less robust when it comes to preserving target peak values.

A convolutional autoencoder is introduced in [18] for noise suppression of the RD spectrum. The prominent target peaks can be preserved after applying the autoencoder used in [18], whereby the phase information and probably the weak target peaks cannot be completely retained.

In this paper, an autoencoder is used for automotive radar interference mitigation and we use a training dataset which is generated by a simulation. For the performance assessment, the trained autoencoder has been applied to real radar measurements. Different from [18], the proposed autoencoder reconstructs interference-contaminated samples in the time domain without suffering from difficulties in the reconstruction of phase. Furthermore, the proposed autoencoder can completely avoid the costly and challenging labeling of the radar sensor data as in [16, 17].

## 2. AUTOMOTIVE FMCW RADAR SIGNAL MODEL

The CS modulation, which is a currently widespread variant of FMCW in automotive radar system, contains L consecutive chirps, and its transmit waveform can be represented as

$$
T _ { x } ( t ) = \sum _ { l = 0 } ^ { L - 1 } x \left( t - l T \right) .\tag{1}
$$

The individual transmit chirp signal is given by

$$
\begin{array} { r } { \boldsymbol { x } ( t ) = \exp \left( j 2 \pi \left( f _ { c } t + 0 . 5 \alpha t ^ { 2 } \right) \right) \operatorname { r e c t } _ { T } ( t ) , } \end{array}\tag{2}
$$

where $f _ { c }$ is the carrier frequency, $\alpha ~ = ~ B / T$ denotes the slope of the transmit signal with $B$ and T denoting the sweep bandwidth and chirp duration, respectively, and $\mathrm { r e c t } _ { T } ( t )$ is the square pulse of duration $T$ . The lth chirp of the target’s echo signal is delayed by $\tau$ from the transmit signal with a normalized relative Doppler shift $d \colon$

$$
r _ { l } ( t ) = A _ { l } x \left( t + ( t + l T ) d - \tau \right) + \mathbf { v } _ { l } ( t ) ,\tag{3}
$$

where $A _ { l }$ is the received amplitude, $\tau = { 2 R } / { c } , d = { 2 v } / { c } ,$ and $\mathbf { v } _ { l }$ denotes the complex Gaussian noise. Here, R and v denote, respectively, the distance and relative radial velocity between the radar and target, c represents the speed of light. The beat signal at the intermediate frequency can be obtained after stretch processing, namely mixing $r _ { l }$ with the complex conjugate of the transmitted signal: $\hat { y } _ { l } ( t ) = r _ { l } ( t ) x ^ { * } ( t )$ . After filtering and sampling with a period of $T _ { s }$ and $N$ samples per chirp, the analogue-to-digital converter (ADC) samples of the discrete beat signal can be approximated as [2]

$$
\begin{array} { r l } & { { \hat { y } } _ { l , n } \approx A _ { l } \mathrm { e x p } \left( j 2 \pi \left( - \alpha \tau + f _ { c } d \right) n T _ { s } \right) } \\ & { \quad \quad \quad \cdot \mathrm { e x p } \left( j 2 \pi \left( f _ { c } d l T \right) \right) + \mathrm { v } _ { l , n } \quad n \in [ 0 , N - 1 ] . } \end{array}\tag{4}
$$

With the assumption that ${ \bar { y } } _ { l , n }$ represents the additive in-band interference, the discrete beat signal can be therefore summarized as

$$
y _ { l , n } = \left\{ \begin{array} { l l } { \hat { y } _ { l , n } + \mathbf { v } _ { l , n } } & { n \subseteq H } \\ { \hat { y } _ { l , n } + \bar { y } _ { l , n } + \mathbf { v } _ { l , n } } & { n \subseteq K , } \end{array} \right.\tag{5}
$$

where H is the set of object reflected samples without interference and K denotes the set of samples containing interference. To obtain the interference-pruned samples, a classical edge detector, e.g., Laplacian filter can be used to identify the anomalies, and these anomalies can be pruned through thresholding [19]. ## 3. RADAR INTERFERENCE MITIGATION

## 3.1. Complex Signal as Network Input Data

The complex beat signal obtained after the ADC sampling is converted into a tensor of the form [N, L, 2] as the input for the encoder network. The samples disturbed by interference are set to zero with a mask. Since the position of interference-contaminated samples can be random, a set of interference-free discrete beat signals in combination with random masks is used as input during training. To ensure that the trained autoencoder is able to deal with the randomness of the interference position, a mask with 0.15N arbitrary positions in every chirp is empirically chosen. The training dataset is simulated based on (4) and organized into 500 batches of four samples each. In order to improve the learning ability and the robustness of the model, the input data is normalized on its maximum after masking.

## 3.2. Autoencoder Network Architecture

An overview of the autoencoder architecture is shown in Fig. 2 which consists of two main components, namely the encoder and decoder networks. The number of kernels of each layer is specified directly above the corresponding layers and can vary during implementation. The encoder network $g _ { \phi }$ extracts the latent features from the residual samples, while the decoder network $f _ { \theta }$ tries to recover the interference-free signal according to the latent features. The gated convolution [20] is used as a fundamental building block of the proposed autoencoder. The convolution stride used in the encoder network is 2.

## 3.3. Gated Convolution

The common way of using conventional CNN is to take all samples from the input data as valid information, that may be suitable for either classification or detection tasks. In signal inpainting tasks, however, not all information from the input data is relevant for the reconstruction of missing areas. Thus the content in the missing areas must be distinguished from the content of the remaining signal. The gated convolution proposed by Yu et al. [20] uses convolution in combination with the sigmoid function so that the network learns this distinction. The network can therefore concentrate on the residual signal to capture the signal pattern and can thus process severely disturbed scenarios that are not contained in the training dataset, as long as the residual signal can reflect the signal pattern in the latent features. The gated convolution is formulated as

$$
\mathbf { G } _ { y , x } = \sum \sum \mathbf { W } _ { g } \cdot { \pmb { I } } \qquad \mathbf { F } _ { y , x }  &  = \sum \sum \mathbf { W } _ { f } \cdot { \pmb { I } }\tag{6}
$$

$$
O _ { y , x } = \xi \left( \mathbf { F } _ { y , x } \right) \odot \sigma \left( \mathbf { G } _ { y , x } \right)\tag{7}
$$

where $\sigma$ is sigmoid function and  denotes the element-wise multiplication. $\xi$ can be any activation function. $\mathbf { W } _ { g }$ and $\mathbf { W } _ { f }$ are two different convolutional filters. $\mathbf { G } _ { y , x }$ and $\mathbf { F } _ { y , x }$ represent the features that are extracted by the convolutional filters $\mathbf { W } _ { g }$ and $\mathbf { W } _ { f }$ , respectively. I is the input signal and O denotes the output signal.

## 3.4. Composite Loss

The recovery losses in the range profile and in the Doppler-FFT spectrum (obtained by applying the FFT to

the discrete beat signal along slow-time samples, namely in the chirp direction) are used as objective function for the proposed autoencoder. The loss function is defined as

$$
\begin{array} { c } { { \displaystyle \epsilon _ { k } = \frac { \gamma } { n _ { s } } \sum _ { s _ { i } \in \Delta } \left| \Phi ( \mathbf { Y } _ { s _ { i } } ^ { k } ) - \Phi ( \tilde { \mathbf { Y } } _ { s _ { i } } ^ { k } ) \right| + \frac { \gamma } { N - n _ { s } } \sum _ { s _ { j } \notin \Delta } \left| \Phi ( \tilde { \mathbf { Y } } _ { s _ { j } } ^ { k } ) \right| } } \\ { { \displaystyle + \frac { 1 - \gamma } { n _ { s } } \sum _ { s _ { i } \in \Delta } \left| \Psi ( \mathbf { Y } _ { s _ { i } } ^ { k } ) - \Psi ( \tilde { \mathbf { Y } } _ { s _ { i } } ^ { k } ) \right| + \frac { 1 - \gamma } { L - n _ { s } } \sum _ { s _ { j } \notin \Delta } \left| \Psi ( \tilde { \mathbf { Y } } _ { s _ { j } } ^ { k } ) \right| } , }  \end{array}\tag{8}
$$

where $\mathbf { Y } _ { s } ^ { k } \in \mathbb { C } ^ { N \times L }$ denotes the k-th interference-free discrete beat signal in each batch, $\tilde { \mathbf { Y } } _ { s } ^ { k }$ the autoencoder-recovered discrete beat signal, Φ denotes the fast Fourier transform along fast-time samples and Ψ is the fast Fourier transform along the chirp direction, $0 \leq \gamma \leq 1$ is the weighting factor, $\Delta$ represents a set of the $n _ { s }$ strongest targets, $0 ~ < ~ n _ { s } ~ <$ min $\{ N , L \}$ and can be chosen empirically. It is found that the autoencoder can efficiently converge by controlling the high-dimensional loss of the recovered discrete beat signal, namely the loss in the range profile and the Doppler-FFT spectrum. It should be noted that the mean absolute error used in (8) can be replaced with the mean squared error and the network delivers similar performance. In our experiments, we found the best choice of $\gamma$ was found at $\begin{array} { r l r } { \gamma } & { { } = } & { 0 . 6 , } \end{array}$ indicating a benefit when the first term in (8) receives more weight. ## 4. PERFORMANCE EVALUATION

In this section, the performance of the autoencoder is evaluated in terms of SINR in comparison with the autoregressive (AR) model and the zeroing method. The zeroing method is a basic technique for interference rejection in which the disturbed samples in the discrete beat signal are simply set to zero. The validation dataset used in this paper is a real radar measurement which is a challenge to the autoencoder because the training dataset contains only the simulated discrete beat signal. The real radar measurement is recorded in a test chamber with radiation-absorbent materials and the targets are created by a target generator.

## 5. CONCLUSION

In this paper, an autoencoder is used as an interference mitigation approach in an automotive radar system. The analysis results show that the proposed method has its advantage in recovering the discrete beat signal compared to the AR and zeroing methods in terms of SINR and the target peak energy. It is shown that even in severely disturbed scenarios, the proposed method can improve the SINR of the RD spectrum by ca. 17 − 24 dB in comparison with the simple zeroing method and up to ca. 18 dB in comparison with the AR method. The proposed method preserves its robustness in discrete beat signal recovery even with up to 55% missing samples, which is much more complex than the training scenarios in which only 15% samples are discarded. Training on real data will be explored in our future work.
