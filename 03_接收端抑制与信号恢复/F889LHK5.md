# SIGNAL RECONSTRUCTION FOR FMCW RADAR INTERFERENCE MITIGATION USINGDEEP UNFOLDING

J. Overdevest<sup>★†</sup>, A.G.C. Koppelaar<sup>★</sup>, M.J.G. Bekooij<sup>★</sup>, J. Youn<sup>†</sup>, R.J.G. van Sloun<sup>†</sup>

Signal Processing Algorithms, NXP Semiconductors, Eindhoven, The Netherlands <sup>†</sup> Dept. of Electrical Engineering, Eindhoven University of Technology, Eindhoven, The Netherlands

## ABSTRACT

Removal of frequency-modulated continuous wave (FMCW) interference by zeroing corrupted samples causes significant distortions and peak power losses in the range-Doppler map. Existing methods aim to diminish these distortions by utilizing data from one dimension to reconstruct the corrupted samples, which do not perform well when a large number of samples are interfered and have dificulty recovering weak target signals.

In this paper, model-based deep learning interference mitigation algorithms, called ALISTA and ALFISTA, are presented that reduce these artifacts by leveraging the full integration gain using all uncorrupted fast-time and slow-time samples. Simulations with 50% corrupted samples show that target peak power loss and velocity peak-to-sidelobe ratio (VPSR) with a 20-layer ALFISTA improves with 5.5 and 9<sup>.</sup>6 dB compared to zeroing. Furthermore, significant improvements in precision and recall are observed, even when large amounts (50-80%) of samples are missing.

Index Terms— Radar Interference Mitigation, Signal Reconstruction, Deep Unfolding, ALISTA, ALFISTA

## 1. INTRODUCTION

Interference across FMCW radars is becoming an important challenge with the number of radar sensors on vehicles with autonomous driving capabilities growing exponentially. Interference occurs when two or more FMCW radars utilize similar time, frequency, and spatial resources. This results in temporary blindness, sensitivity losses, and ghost targets. Solutions are being developed to mitigate radar interference in both the analog and digital domain, as well as coordination strategies to avoid its occurrence [1, 2, 3].

Typically, radar-to-radar interference is handled in the digital domain. After detection of the interfered samples, the interference can be removed in the time domain by substituting the corrupted samples with: zeros [4], reconstructed samples from the Iterative Method with Adapted Thresholding (IMAT) [5], or from an autoregressive model applied to the slow-time samples [6]. Another approach is to convert the beat signal to the time-frequency domain or wavelet domain to better characterize and separate the interfered signal from the target signal. Neemat et al. introduced interference detection in time-frequency domain, using the short-time Frequency transform (STFT), where the corrupted samples are substituted with interpolated samples [7]. Recently, deep learning methods have been developed to mitigate interference. In [8], a time-domain reconstruction method is initiated using a Bi-RNN model with a gated recurrent unit.

In this paper, we introduce a model-based deep learning method that exploits the locations of the corrupted samples, i.e., the detection mask, by casting interference mitigation as a compressive recovery problem. Besides, we proposed to learn the update rate and shrinkage parameters which are diferent for every unfolding. This resulted in improved convergence performance and less computational complexity, compared to classical (F)ISTA with fixed parameters, which is required for real-time automotive radar sensing. Furthermore, the proposed two-dimensional time-domain signal reconstruction can cope with long intervals of corrupted samples. Due to the 2D coherent processing gain, it can reconstruct weak targets better compared to the aforementioned 1D state-of-the-art techniques. Generally, these methods operate on individual groups of either fast-time samples or slow-time samples, which fail if a large number of samples are corrupted consecutively along one or both axes.

## 2. PROBLEM FORMULATION

Detecting interfered samples can be done in the time domain [9]. The detection result can be described with a binary interference detection mask m ∈ {0<sup>,</sup> 1}<sup>??×??</sup> , which denote the corrupted and clean samples. Here, <sup>??</sup> and <sup>??</sup> refer to the number of samples within an FMCW chirp (fast-time samples) and the number of FMCW chirps (slow-time samples), respectively. Subsequently, the impact of FMCW interference can be reduced by replacing the corrupted samples with zeros as illustrated with the following expression:

$$
\mathbf { y } = \mathbf { m } \odot \mathbf { x } .\tag{1}
$$

Here, $\mathbf { x } \in \mathbb { R } ^ { M \times N }$ is the beat signal and $\mathbf { y } \in \mathbb { R } ^ { M \times N }$ is the interference-mitigated beat signal using zeroing, and ⊙ denotes the element-wise multiplication. Zeroing causes the coherent processing gain to be reduced because of the substituted zeros, as well as a phase distortion. Both can be explained by the frequency convolutional theorem

$$
{ \mathbf { x } } _ { m , : } \odot  { \mathbf { m } } _ { m , : }   { \mathbf { X } } _ { m , : } \circledast  { \mathbf { M } } _ { m , : } , \mathrm { ~ } \mathrm { w i t h }\tag{2}
$$

$$
\mathbf { M } _ { m , : } = T \mathrm { s i n c } ( \pi T f ) - \sum _ { i } T _ { m , i } \mathrm { s i n c } ( \pi T _ { m , i } f ) e ^ { j 2 \pi f \tau _ { m , i } }\tag{3}
$$

Here, <sup>??</sup> is the chirp duration, and $T _ { m , i }$ and $\tau _ { m , i }$ are the duration and center position of the $i ^ { \mathrm { { t h } } }$ interference interval in the $m ^ { \mathrm { t h } }$ chirp index, defined from rect $\cdot \frac { t - \tau _ { m , i } } { T _ { m , i } } )$ function, i.e., relating to the $i ^ { \mathrm { { t h } } }$ gap in the mask. This entails that the beat frequency response of every radar target in $\mathbf { X } _ { m , \astrosun }$ <sub>:</sub> is convolved with ${ \bf { M } } _ { m , : } ,$ introducing unique amplitude and phase distortions in the $m ^ { \mathrm { { { t h } } } }$ range Fourier Transform output.

## 3. METHODOLOGY

Like the authors in [10], we formulate the signal recovery as a sparse compressive sensing problem, such as LASSO, that specifically penalizes the <sup>ℓ</sup> -norm via the shrinkage scalar <sup>??</sup> by

$$
\hat { \mathbf { x } } = \arg \operatorname* { m i n } _ { \bar { \mathbf { x } } } \frac { 1 } { 2 } | | \bar { \mathbf { y } } - \mathbf { A } \bar { \mathbf { x } } | | _ { 2 } ^ { 2 } + \lambda | | F \{ \mathbf { x } \} | | _ { 1 } \ ,\tag{4}
$$

with $F \{ . \}$ and $F ^ { - 1 } \{ . \}$ denoting the 2D Fourier and inverse Fourier transform, respectively. The measurement matrix A ∈ $\{ 0 , 1 \} ^ { P \times N M }$ is derived from the 2D binary detection mask with the number of rows $P < < M N$ , and y¯ and x¯ refer to the zero-mitigated analog-to-digital converted (ADC) samples and non-interfered ADC samples, vectorized from y and x. Each row of A is constrained to having cardinality one. The optimization problem of (4) can be solved using the iterative shrinkage and thresholding algorithm (ISTA)

$$
\bar { \mathbf { x } } ^ { k + 1 } = F ^ { - 1 } \left\{ \mathcal { T } _ { \lambda } \left( F \left\{ \bar { \mathbf { x } } ^ { k } \right\} - F \left\{ \mu \mathbf { A } ^ { H } \left( \mathbf { A } \bar { \mathbf { x } } ^ { k } - \bar { \mathbf { y } } \right) \right\} \right) \right\} \mathrm { ~ . ~ }\tag{5}
$$

Here, $\mu$ is the update step size and $\mathcal { T } _ { \lambda }$ is the soft thresholding function. Generally, ISTA requires careful tuning of <sup>??</sup> and $\mu ,$ as it plays a significant role in the convergence of the solution.

## 3.1. ISTA for Interference Mitigation

To avoid the use of large vectors when deploying the linear model $\bar { \mathbf { y } } = \mathbf { A } \bar { \mathbf { x } }$ , we introduce a generalized ISTA form that replaces the linear model with an element-wise multiplication and exploit the sparsity of the range-Doppler domain with a 2D Fourier transform:

$$
F \{ \mathbf { y } \} = F \{ \mathbf { m } \odot \mathbf { x } \}\tag{6}
$$

Using (6), we reformulate the ISTA algorithm for the linear model denoted in (5) such that it can be applied to twodimensional interference mitigation

$$
\mathbf { x } ^ { k + 1 } = F ^ { - 1 } \left\{ \mathcal { T } _ { \lambda } \left( F \left\{ \mathbf { x } ^ { k } \right\} - \mu \left( F \left\{ \mathbf { m } \odot \mathbf { x } ^ { k } \right\} - F \left\{ \mathbf { y } \right\} \right) \right) \right\}\tag{7}
$$

Additional rewriting and taking into account the linearity property in the Fourier Transform leads to the following complexity-reduced form

$$
\mathbf { x } ^ { k + 1 } = F ^ { - 1 } \left\{ \mathcal { T } _ { \lambda } \left( F \left\{ \left( \mathbf { 1 } - \mu \mathbf { m } \right) \odot \mathbf { x } ^ { k } + \mu \mathbf { y } \right\} \right) \right\}\tag{8}
$$

where $\mu$ determines to what extent the clean, original samples are altered in the reconstruction of the missing samples. Compared to traditional ISTA, a complex soft thresholding function $\mathcal { T } _ { \lambda } ( \mathbf { x } ) = e ^ { j \angle \mathbf { x } } ( | \mathbf { x } | - \lambda )$ <sub>+</sub> was used. For comparison purposes with state-of-the-art methods, all results in this paper have been obtained with $\mu = 1$

Unlike [10], we propose a solution that utilizes all chirp data to reconstruct the missing samples, while they base their reconstruction on specific subsets of uncorrupted samples and focus on cases with a limited amount of corrupted samples (up to 15%). Besides, we deploy deep learning to achieve higher convergence rates.

In the next section, we explain how we apply algorithm unrolling using the data consistency step of (8) to improve the convergence rate compared to conventional iterative algorithms while showing that the learned approach is still capable of reconstructing large amounts of missing data.

## 3.2. Deep Unfolding

In the last decade, deep unfolding has gained substantial interest. It steps away from conventional black-box approaches and has a close connection to iterative algorithms, which helps better interpretability and the networks to better generalize. In deep unfolding, each ISTA iteration maps to a single network layer of which many layers are stacked [11].

Analytic Learned ISTA (ALISTA) is a deep unfolded variant of ISTA, proposed in [12], where the analytic form is contained. For interference mitigation, the analytic model is desirable as the subsampling mask m is known from the interference detection stage and can be exploited for reconstructing the missing beat signal samples. For each $k ^ { \mathrm { t h } }$ layer in ALISTA, the step size $\mu _ { k }$ and shrinkage threshold $\lambda _ { k }$ replace the fixed $\mu$ and <sup>??</sup> of (8) as for ISTA.

## Analytic Learned Fast ISTA

Fast ISTA (FISTA) preserves the computational simplicity of ISTA while the rate of convergence is significantly better due to an additional momentum term [13]. We propose a model-based deep unfolding method, called Analytic Learned FISTA (ALFISTA). ## 3.3. Training Details

AL(F)ISTA is optimized using a training set of size 1<sup>,</sup> 000, containing various synthetically generated interference scenarios. The parameters $\mu$ and <sup>??</sup> are learned by minimizing the mean squared error between the model’s output $\hat { \mathbf { x } }$ and the clean ADC samples x. It may be beneficial to train the unfolded network dedicated for a specific interference severeness, e.g., for a given number of zeros $N _ { 0 }$ present in ADC samples of a single measurement, i.e., a distortion ratio $r = N _ { 0 } / ( N M )$ · 100%. During inference, the learned parameters $\mu$ and <sup>??</sup> are preloaded. Hence, these unfolded networks have an extremely low memory footprint due to the low number of weight parameters that need to be stored.

## 4. RESULTS

## 4.1. Simulation Setup

We simulate $L \in [ 2 , \ldots , 1 5 ]$ targets with varying radar cross section $\in [ - 8 , 3 0 ]$ dBsm where the distance and velocity of each target are randomly chosen within the unambiguous intervals. The interference signals are generated with randomized chirp slopes and pulse repetition intervals (PRI) resulting in both correlated and uncorrelated interferers, for which the number of interferers is increased to reach higher distortion ratios.

## 4.3. Evaluation

Fig. 2 presents the range-Doppler outputs of the clean signal, zeroed signal, IMAT, ALISTA, and ALFISTA methods with $K = 2 0$ for distortion ratios of $r = 2 0 \%$ and $r = 5 0 \%$ respectively. The distortions caused by zeroing cannot be reduced by IMAT as the reconstruction capabilities of the 1D iterative approach degrade when many consecutive samples are zeroed. ALISTA shows improvements in signal recovery, but especially distortions that arise from strong targets are not completely removed. The improved convergence of ALFISTA shows its superiority compared to the other methods, even when $r = 5 0 \%$ . Instead of evident sidelobes, due to periodicity in the zero locations, as depicted in Fig. 2a, Fig. 2b contains less prominent sidelobes due to a higher number of zeros that are more randomized. However, ALFISTA is the only method that can reduce the artifacts, such that the two targets at $R = 2 0 \mathrm { m }$ are distinguished. Furthermore, all presented reconstruction methods improve the target peak power, but not the thermal noise floor.

In Fig. 3, an example of a chirp is shown where all samples are corrupted and replaced with zeros when the overall distortion rate is $r = 7 0 \%$ . The proposed 2D signal recovery methods utilize the data from other chirps to accurately reconstruct the missing samples, even with relatively few clean samples available in the full measurement.

The aforementioned methods are compared for increasing <sup>??</sup> in Fig. 4 using the metrics defined in Section 4.2. AL-ISTA and ALFISTA are compared for multiple layers, namely $K = 2 , 5 , 1 0 , 2 0 , 5 0$ . From Fig. 4a-c, one can observe that IMAT does not significantly improve compared to zeroing, as the number of zeros in fast-time is too large. Both have significantly low VPSR and PSNR, and high $\mathrm { P _ { l o s s } }$ . Consequently, the reduced peak power and VPSR lead to high FP and FN with lower precision and recall as a result for increasing <sup>??</sup> as shown in Fig. 4d-e. When applying ALISTA and ALFISTA, we are able to partially restore the target peak power and VPSR, and therefore, leading to increased performance in precision and recall. As <sup>??</sup> increases, the reconstruction quality is improved. This is shown in the PSNR curves (Fig. 4c), which signify reduced mean squared error although the thermal noise floor will never be restored when it is compared to the clean image. Finally, when $r = 5 0 \%$ , on average 9<sup>.</sup>6 dB and 5<sup>.</sup>5 dB improvements are achieved in restoring the peak power and VPSR respectively for ALFISTA with <sup>??</sup> = 20 compared to zeroing.

## 5. CONCLUSIONS

We presented deep unfolding methods, ALISTA and AL-FISTA, that reconstruct the corrupted time-domain samples by leveraging the sparsity attribute of radar targets in the range-Doppler plane while utilizing all uncorrupted data across the fast-time and slow-time dimensions. Although ALFISTA achieves higher performance, ALISTA might be beneficial when limited computing power and memory is available. Quantitative results show on average 5<sup>.</sup>5 and 9<sup>.</sup>9 dB improvements in target peak power and VPSR for all simulated targets when 50% of the samples were missing. The proposed methods provide a promising way to mitigate interference as the number of radars deployed for autonomous driving applications grows in the future.
